About

Why Port.Scan Exists

A small, honest tool for looking at what's listening on a network.

The Project

Port.Scan started as a way to learn how port scanning actually works under the hood, and to have a clean, self-hosted interface for running scans on my own home network — instead of reaching for a heavier, closed-source tool.

It's intentionally small: a Flask app, a Python scanning core, and a single page interface. No accounts, no cloud dependency, no data leaving your machine.

Built With

Python 3 Flask HTML / CSS Vanilla JS

Scanning is handled with Python's standard library socket module and a thread pool for concurrency; results are persisted to a lightweight local database.
